🗃️ BD OperacionalOracle / PostgreSQL
Analítica / IA
🤖 Motor ML/IAScoring crediticio
📊 BI EjecutivoPower BI / Tableau
🔮 Analítica PredictivaMorosidad / churn
📡 Reportería SBSAutomatizada
Estrategia de Integración de Datos
🚌
Event-Driven (Kafka) Para datos en tiempo real: transacciones, pagos, alertas LAFT. Latencia <1 segundo.
🔄
ETL Batch (Apache NiFi) Para reportes nocturnos SBS, carga de DWH, sincronización RRHH y contabilidad.
🔗
APIs REST/GraphQL Para integración con Reniec, Equifax, SBS, BIM y canales digitales. Versionadas y documentadas.
🗂️
MDM — Dato Único Master Data Management para cliente, producto y agencia. Un solo registro de verdad (Single Source of Truth).
Políticas de Gobierno del Dato
👑
Data Ownership Cada entidad tiene un propietario funcional responsable de la calidad y actualización del dato.
🔒
Clasificación de Datos 4 niveles: Público / Interno / Confidencial / Restringido (datos cliente = Confidencial).
📏
Estándares de Calidad KPIs de calidad: completitud ≥95%, consistencia ≥98%, disponibilidad 99.9% para Core.
🛡️
Privacidad (LPDP) Cumplimiento Ley 29733 (Protección Datos Personales). Consentimiento explícito del cliente.
📖
Catálogo de Datos Implementar Apache Atlas o Collibra para descubrimiento y linaje de datos corporativos.

Sesión 4 · Producto DArquitectura de Infraestructura TO-BE — Cloud Híbrido + Ciberseguridad

Modelo de Despliegue: On-Premise + Cloud Híbrido + SaaS

🏢 On-Premise (Núcleo)

  • Core Bancario crítico (misión)
  • Base de datos Oracle transaccional
  • Módulo LAFT y cumplimiento SBS
  • Servidores HP Gen11 (actualizados)
  • Data Center Lima + DR Arequipa
  • Red LAN segmentada por VLANs
  • Firewall next-gen (Fortinet 200F)

☁️ Cloud Híbrido (Azure/AWS)

  • Data Lake (Azure Data Lake Gen2)
  • Data Warehouse (Snowflake)
  • App móvil backend (Kubernetes)
  • API Gateway (Azure APIM)
  • Motor IA/ML (Azure ML Studio)
  • Backup cloud (Azure Backup)
  • SIEM en cloud (Microsoft Sentinel)
  • SD-WAN gestionado (cloud-managed)

🌐 SaaS (Servicios Externos)

  • Microsoft 365 (correo + Teams)
  • Workday / BambooHR (HRMS)
  • ServiceNow (ITSM)
  • Salesforce / HubSpot (CRM)
  • Moodle Cloud (LMS)
  • DocuSign (firma electrónica)
  • Power BI Premium (BI ejecutivo)
Estrategia de Ciberseguridad — Modelo Defense in Depth
🔴
Capa 1 — PerímetroFirewall Fortinet 200F Next-Gen · IDS/IPS · WAF (Web Application Firewall) · DDoS Protection · Filtro DNS (Cisco Umbrella)
🔐
Capa 2 — Identidad y Acceso (IAM)Zero Trust Architecture · MFA obligatorio para todos los usuarios · SSO corporativo (Azure AD) · PAM para accesos privilegiados (administradores)
💻
Capa 3 — Endpoint ProtectionEDR (CrowdStrike / Defender for Endpoint) · Control de dispositivos · Cifrado de disco (BitLocker) · Gestión MDM de tablets asesores
🔎
Capa 4 — Monitoreo y Detección (SIEM)Microsoft Sentinel · Correlación de eventos 24/7 · SOAR para respuesta automática a incidentes · Gestión de logs centralizada · Alertas LAFT automatizadas
🛡️
Capa 5 — Continuidad y Recuperación (BCP/DRP)RTO ≤4h · RPO ≤1h para Core Bancario · Backup cloud en Azure · Data Center secundario (DR) en Arequipa · Pruebas de recuperación semestrales
